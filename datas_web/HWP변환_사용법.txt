📄 HWP → TXT 자동 변환 도구 사용법

========================================
🎯 목적
========================================

HWP 파일을 PDF 뷰어에서 TTS로 읽을 수 있도록
페이지별로 구분된 텍스트 파일(.txt)로 자동 변환합니다.


========================================
⚙️ 설치 (최초 1회만)
========================================

1. "install_hwp_tool.bat" 더블클릭
2. 자동으로 필요한 라이브러리(olefile) 설치
3. 완료!


========================================
🚀 사용 방법 (매우 간단!)
========================================

방법 1: 드래그 앤 드롭
  1. HWP 파일을 "hwp_convert.bat"에 드래그 앤 드롭
  2. 자동으로 같은 폴더에 .txt 파일 생성!

방법 2: 명령줄
  hwp_convert.bat [HWP파일경로]

방법 3: Python 직접 실행
  python hwp_to_txt.py [HWP파일경로]


========================================
📝 출력 형식
========================================

자동으로 다음 형식의 .txt 파일이 생성됩니다:

=== 페이지 1 ===
첫 번째 페이지 텍스트 내용...

=== 페이지 2 ===
두 번째 페이지 텍스트 내용...

=== 페이지 3 ===
세 번째 페이지 텍스트 내용...


========================================
🎬 PDF 뷰어에서 사용하기
========================================

1. PDF 파일을 PDF 뷰어에서 로드
2. 설정(⚙) 버튼 클릭
3. "📄 텍스트 파일에서 불러오기" 섹션 찾기
4. 변환된 .txt 파일 선택
5. "텍스트 파일 적용" 클릭
6. "자동 재생 시작" → TTS로 재생! 🎉


========================================
💡 예시
========================================

HWP 파일: C:\Documents\수업자료.hwp
  ↓ 변환
TXT 파일: C:\Documents\수업자료.txt
  ↓ PDF 뷰어에서 업로드
자동 TTS 재생! 🔊


========================================
⚠️ 주의사항
========================================

- HWP 5.0 이상 버전만 지원
- HWP 파일에 텍스트가 있어야 추출 가능
- 이미지만 있는 HWP는 변환 불가
- 복잡한 레이아웃은 일부 텍스트 순서가 바뀔 수 있음


========================================
🆘 문제 해결
========================================

Q: "pip를 찾을 수 없습니다" 오류
A: Python이 설치되지 않았습니다. Python 3.x 설치 필요

Q: 텍스트가 깨져서 나옴
A: HWP 파일 인코딩 문제. 한글 프로그램에서 다시 저장 후 시도

Q: 변환은 되는데 내용이 비어있음
A: HWP 파일에 텍스트가 없거나 이미지만 있는 파일
