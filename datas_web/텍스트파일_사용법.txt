📄 텍스트 파일 사용법

## HWP 파일을 TXT로 변환하는 방법

1. HWP 파일을 한글 프로그램으로 엽니다
2. "파일" > "다른 이름으로 저장" 클릭
3. "파일 형식"에서 "텍스트 문서 (*.txt)" 선택
4. 저장 클릭

## 텍스트 파일 형식

각 페이지는 다음과 같은 형식으로 구분합니다:

=== 페이지 1 ===
첫 번째 페이지의 텍스트 내용

=== 페이지 2 ===
두 번째 페이지의 텍스트 내용

=== 페이지 3 ===
세 번째 페이지의 텍스트 내용

...

## 중요 사항

- 각 페이지 구분자는 반드시 "=== 페이지 숫자 ===" 형식이어야 합니다
- 페이지 번호는 숫자만 입력합니다 (예: 1, 2, 3...)
- 페이지 번호는 PDF의 실제 페이지 번호와 일치해야 합니다
- 빈 페이지는 건너뛰어도 됩니다

## 사용 방법

1. PDF 뷰어에서 PDF 파일을 로드합니다
2. 설정(⚙) 버튼을 클릭합니다
3. "텍스트 파일에서 불러오기" 섹션을 찾습니다
4. "파일 선택" 버튼을 클릭하여 변환한 .txt 파일을 선택합니다
5. TTS 재생 후 대기 시간을 설정합니다 (기본값: 2초)
6. "텍스트 파일 적용" 버튼을 클릭합니다
7. "자동 재생 시작" 버튼을 눌러서 TTS로 재생합니다
