🌐 HWP 변환 웹 서버 사용법

========================================
🚀 초간단 사용법
========================================

1. "start_hwp_server.bat" 더블클릭
2. 자동으로 서버 시작됨
3. 브라우저에서 http://localhost:5000 열기
4. HWP 파일 드래그 앤 드롭
5. 자동 변환 완료!
6. TXT 파일 다운로드


========================================
💡 장점
========================================

✅ 브라우저에서 바로 사용
✅ 드래그 앤 드롭 간편 업로드
✅ 자동으로 페이지별 구분
✅ 변환 결과 미리보기
✅ TXT 파일 즉시 다운로드


========================================
🔄 전체 흐름
========================================

HWP 파일 준비
  ↓ 드래그 앤 드롭
웹 서버에서 자동 변환
  ↓ TXT 다운로드
PDF 뷰어에 업로드
  ↓
TTS 자동 재생! 🎉


========================================
📝 예시
========================================

1. start_hwp_server.bat 실행
2. 브라우저 자동 열림 (또는 http://localhost:5000 접속)
3. HWP 파일 드래그 앤 드롭
4. "📥 TXT 파일 다운로드" 클릭
5. PDF 뷰어에서 다운로드한 TXT 업로드
6. 끝!


========================================
⚙️ 추가 기능
========================================

- 2가지 HWP 파싱 엔진 자동 시도
  1) hwp5tools (더 정확함)
  2) olefile (더 빠름)

- 변환 실패시 자세한 오류 메시지 표시
- 긴 텍스트는 미리보기만 표시
- UTF-8 인코딩으로 자동 저장


========================================
🆘 문제 해결
========================================

Q: 서버가 시작되지 않음
A: Python이 설치되어 있는지 확인

Q: "포트 5000을 사용할 수 없습니다" 오류
A: 다른 프로그램이 5000 포트를 사용 중. 그 프로그램 종료 후 재시도

Q: 변환은 되는데 텍스트가 이상함
A: HWP 파일 자체의 문제. 한글 프로그램에서 다시 저장 후 시도

Q: 0개 섹션 추출
A: 이 웹 서버 방식을 사용하면 해결됨!
