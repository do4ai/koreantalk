# PDF 자동 재생 시나리오 설계

본 문서는 KoreanTalk PDF 뷰어의 자동 재생 시퀀스와 오디오 출력 설계를 정의합니다.

## 1. 오디오 리소스 위치
- **질문 시작 알림음**: `./data/sound_qstart.mp3`
- **말하기 시작 신호음**: `./data/sound_speak.mp3`

## 2. 자동 재생 프로세스 (페이지별)

각 페이지로 이동 시 재생 버튼을 누르면 다음 순서로 동작합니다:

| 단계 | 동작 | 설명 | 표시 메시지 |
| :-- | :-- | :-- | :-- |
| **1단계** | **안내음 재생** | `./data/sound_qstart.mp3`를 재생하여 문제 시작을 알림 | 안내 시작 |
| **2단계** | **본문 낭독 (TTS)** | 해당 페이지에 설정된 텍스트를 한국어 음성으로 출력 | 본문 낭독 중 |
| **3단계** | **준비 시간 (대기)** | 사용자가 답변을 준비하는 시간 (설정된 초만큼 카운트다운) | 준비 시간 |
| **4단계** | **시작 신호음** | `./data/sound_speak.mp3`를 재생하여 답변 시작을 유도 | 신호 대기 |
| **5단계** | **말하기 시간** | 사용자가 답변을 말하는 실전 시간 (설정된 초만큼 카운트다운) | 말하기 시간 |

## 3. 예외 처리 및 제어
- **중지**: 재생 중 언제든지 중지 버튼을 누르면 TTS와 타이머가 즉시 정지됩니다.
- **페이지 이동**: 이전/다음 페이지 이동 시 현재 재생 중인 시퀀스를 중단하고 새 페이지의 1단계부터 시작합니다.
- **반복**: 마지막 단계 완료 후 '반복 재생'이 켜져 있다면 동일 페이지의 1단계로 돌아갑니다.
- **자동 넘김**: '반복 재생'이 꺼져 있다면 다음 페이지로 자동 이동합니다.
