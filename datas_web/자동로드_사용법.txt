🚨 중요: 자동 텍스트 로드를 사용하려면 반드시 HTTP 서버를 실행해야 합니다!

========================================
⚠️ 문제: file:// 프로토콜 제한
========================================

브라우저에서 직접 HTML 파일을 열면(file://...)
보안상의 이유로 다른 로컬 파일(텍스트 파일)을 읽을 수 없습니다.

❌ 작동 안 함:
file:///C:/Users/Administrator/claude/koreantalk/pdf-viewer.html

========================================
✅ 해결 방법: HTTP 서버 사용
========================================

1. "start_web_server.bat" 더블클릭
2. 검은 창이 뜨고 서버가 시작됨
3. 브라우저에서 다음 주소 열기:

   http://localhost:8000/pdf-viewer.html

   또는

   http://localhost:8000/pdf-manager.html

4. 이제 자동 텍스트 로드가 작동합니다! ✅

========================================
📁 파일 준비
========================================

PDF 파일: "토픽 말하기 최종수정본 25.11.06.pdf"

텍스트 파일들을 hwp_pages 폴더에 준비:
- hwp_pages/토픽 말하기 최종수정본 25.11.06_page1.txt
- hwp_pages/토픽 말하기 최종수정본 25.11.06_page2.txt
- hwp_pages/토픽 말하기 최종수정본 25.11.06_page3.txt
- ...

========================================
🎯 전체 순서
========================================

1. start_web_server.bat 실행 (한 번만)
2. 브라우저에서 http://localhost:8000/pdf-viewer.html 열기
3. PDF 파일 선택
4. 자동으로 텍스트 파일 로드! ✨
5. 자동 재생 시작!

========================================
💡 팁
========================================

- 서버는 한 번만 실행하고 계속 사용 가능
- 서버 창은 닫지 말 것
- 여러 PDF를 계속 열 수 있음
- 콘솔(F12)에서 로드 상태 확인 가능

========================================
🔍 콘솔 로그 확인
========================================

브라우저에서 F12 누르고 Console 탭에서:
✅ 페이지 1: 토픽 말하기 최종수정본 25.11.06_page1.txt 로드 완료
✅ 페이지 2: 토픽 말하기 최종수정본 25.11.06_page2.txt 로드 완료
...

========================================
🆘 문제 해결
========================================

Q: 여전히 로드 안 됨
A: F12 눌러서 Console 탭에서 빨간 에러 메시지 확인

Q: 파일명이 정확한지 확인하는 방법?
A: 콘솔에서 "페이지 X 시도: 파일명.txt" 로그 확인

Q: 서버가 이미 실행 중이라는 오류
A: 정상입니다. 이미 실행 중이면 계속 사용하세요.
