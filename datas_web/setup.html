Google Cloud TTS API 설정

비용 정보: 월 400만 글자 무료, 이후 $4/100만 글자 (매우 저렴!)

1단계: Google Cloud 프로젝트 생성

  1. Google Cloud Console 열기
  2. 새 프로젝트 생성 또는 기존 프로젝트 선택
  3. 결제 계정 연결 (무료 평가판 $300 크레딧 사용 가능)

2단계: Text-to-Speech API 활성화

  1. Text-to-Speech API 페이지 열기
  2. "사용 설정" 버튼 클릭
  3. API가 활성화될 때까지 대기 (1-2분)

3단계: API 키 생성

  1. 사용자 인증 정보 페이지 열기
  2. "+ 사용자 인증 정보 만들기" 클릭
  3. "API 키" 선택
  4. 생성된 API 키를 복사 (나중에 다시 확인 가능)

4단계: API 키 설정

js/config.js 파일을 열고 다음과 같이 수정하세요:

const GOOGLE_TTS_CONFIG = { apiKey: '여기에_복사한_API_키_붙여넣기', endpoint: 'https://texttospeech.googleapis.com/v1/text:synthesize' };
참고: API 키를 공개 저장소에 올리지 마세요!

5단계: 테스트

  1. index.html 파일을 브라우저에서 열기
  2. 아무 Chapter 선택
  3. "🔊 듣기" 버튼 클릭하여 음성 재생 확인
설정 완료! 시작하기 →

문제 해결

  • "API 키가 설정되지 않았습니다" → config.js 파일에 API 키를 올바르게 입력했는지 확인
  • "음성 생성에 실패했습니다" → API가 활성화되어 있고 API 키가 유효한지 확인
  • 음성이 재생되지 않음 → 브라우저 콘솔(F12)에서 오류 메시지 확인

더 자세한 정보는 README.md 파일을 참고하세요.
