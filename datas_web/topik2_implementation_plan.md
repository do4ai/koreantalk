# TOPIK 2 교재 관련 기능 구현 단계 정리

제안해주신 TOPIK II (듣기·읽기, 쓰기, 문법·어휘) 기능을 **'먼저 구현할 것(현 단계)'**과 **'나중에 구현할 것(고도화 과제)'**으로 분류하였습니다. 
특히 먼저 구현할 기능은 현재까지 개발된 내역을 바탕으로 **'이미 구현(혹은 UI가 마련)된 것'**과 **'앞으로 단기간에 구현하기 쉬운 것'** 순으로 재정렬하였습니다.

---

## 1. 먼저 구현할 것 (현 단계 구현/전환 범위)

현재 개발 중인 웹 기반 PDF 뷰어/학습 플랫폼(`pdf-viewer.html` 등)을 기준으로 단기적으로 확실하게 완성하고 서비스할 수 있는 핵심 기능들의 상세 내용입니다.

### 1) 현재 완전히 혹은 대부분 구현되어 바로 서비스 전환이 가능한 기능
현재 시스템의 기본 구조(PDF 뷰어 엔진, 사이드 패널, 컨트롤 바 등)에 이미 로직이 포함되어 있어 즉시 활용 가능한 기능들입니다.

* **웹 기반 클라우드 전자책 변환 (교재 공통)**
  * **상세:** 기존 한글(HWP), 워드(DOCX), PDF 교재(쓰기, 문법·어휘 등)를 앱 설치 없이 브라우저(Web) 환경에서 즉시 띄울 수 있도록 자체 PDF 뷰어를 구축했습니다.
  * **구현 상태:** `pdf-viewer.html`을 통해 PDF 파일 목록 불러오기 및 페이지 이동, 전체 화면 전환이 가능합니다.

* **지문 및 도판 확대·축소 (쓰기 및 공통)**
  * **상세:** 모바일과 태블릿 환경에서 자료 제시형(표, 그래프, 안내문, 이미지 등) 지문을 학습자가 편하게 읽도록 지원합니다.
  * **구현 상태:** 뷰어 하단 컨트롤 바에 확대(+), 축소(-), 너비 맞춤(↔) 버튼 체계가 이미 구현되어 작동 중입니다.

* **오디오 및 음성 제공 (듣기/말하기)**
  * **상세:** 듣기 지문 내용을 음성 형태(TTS/오디오)로 재생하여 학습자가 실전처럼 들을 수 있게 합니다.
  * **구현 상태:** 말하기(Speaking) 영역의 경우 TTS 세팅 기능, 자동 재생, 속도 조절(0.5x~2.0x), 준비/말하기 타이머 바가 모두 연동 완료되어 있습니다. 

* **문항 마킹 및 화면 흐름 제공 (듣기·읽기)**
  * **상세:** 듣기나 읽기 시 좌측에는 지문을 띄우고 우측이나 하단에는 터치 방식으로 답안을 마킹할 수 있는 인터페이스(OMR)를 제공합니다.
  * **구현 상태:** 듣기/읽기(Exam) 탭이 개설되어 있으며, PDF 페이지를 기반으로 문항(1~4번 선택지 등) 선택이 가능한 OMR 모달 편집/추가 UI가 설계되어 있습니다.

### 2) 앞으로 단기간에 구현하기 쉬운 것 (약간의 데이터/로직 연결만 남은 기능)
이미 화면 UI 구조는 완성되어 있으며, DB 연결이나 단순 판별 로직(채점)만 추가하면 완성되는 기능들입니다.

* **정오 판단 기반 단순 자동 채점 (듣기·읽기)**
  * **상세:** 학습자가 우측 OMR 패널에 표기한 답안(예: 1번 문항 ②번 답)을 제출하면, 사전에 엑셀이나 DB에 입력해 둔 '정답지 데이터'와 단순 비교하여 맞고 틀림(O/X)을 즉각 판단하고 점수를 산출합니다.
  * **구현 난이도 하:** 복잡한 인공지능이 필요 없이 `사용자 답안 == 정답` 여부만 판별하면 되므로, 단기간 내에 채점 로직을 연동하여 결과 페이지를 보여줄 수 있습니다.

## 2. 나중에 구현할 것 (향후 R&D 및 데이터바우처 고도화 과제)

데이터 축적, AI 알고리즘 고도화, 타 시스템(LMS)과의 강력한 연동이 필수적인 중장기 연구개발 과제들입니다.

### 1) 진단 및 처방형(설명형) 피드백 (공통 고도화)
* **단순 정답 확인을 넘은 오답 분석**: "틀렸는지 여부"를 넘어 "왜 틀렸는지", "무엇을 보완해야 하는지"를 알려주는 설명형 채점 체계 구축
* **쓰기 오류 유형 진단**: 쓰기 문항 제출 시 오류 유형을 파악하고 향후 개선 방향(수정 가이드, 다음 학습 제안)을 제시하는 처방형 피드백 제공

### 2) 데이터 기반 개인화 학습 경로 (공통 고도화)
* **등급별/수준별 학습 경로 자동 생성**: 누적된 학습 로그와 성취도를 기반으로 1~6급 등급별 과정 구성
* **취약점 기반 자동 조정**: 오답 패턴 및 학습자의 목표 등급 등을 반영해 학습 순서와 복습 계획을 자동 추천(LMS 연동)
* **문법·어휘 콘텐츠 타기팅**: 수준별(3급~6급)로 체계적으로 체계화하여 적절한 난이도 콘텐츠 매칭

### 3) 생성형 AI 및 고성능 평가 모델 도입
* **다중모달 자동채점 (듣기/말하기)**: 음성·텍스트 등 다중 입력 신호를 활용해 발음, 억양, 유창성 등 실전 말하기 역량을 신뢰성 있게 자동 채점
* **쓰기 자동채점 고도화**: 단순 글자 수나 단어 매칭 수준을 넘어 더 정교한 평가가 가능한 최신 AI 기술 기반 모델 도입
* **AI 예문/연습 문항 자동화**: 생성형 AI를 활용하여 학습 단계에 맞게 실전 문장 맥락 예문과 반복 연습용 문항을 자동 생성

### 4) 글로벌 접근성 및 클라우드 안정성 확보
* **다국어 교과서형 현지화 체계**: 해외 학습자를 위해 단순 번역을 넘어 설명·피드백·예시 체계까지 반영한 국가별 현지화 구축
* **운영·보안·저작권 강화**: 개인정보 및 콘텐츠 저작권 보호, 다수 유저 트래픽을 대응하기 위한 클라우드 서비스 운영 안정성 프로세스 정립
